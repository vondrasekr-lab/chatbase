# SQL Guidelines for Antigravity

Vždy dodržuj tento formát při vytváření nebo úpravě databáze:

1. **Code Block v chatu**: SQL kód musí být vždy uveden přímo v odpovědi v markdownovém bloku ` ```sql ` tak, aby mohl uživatel použít tlačítko "Copy".
2. **Odkaz na soubor**: Pod code blokem musí být vždy klikatelný odkaz na fyzický `.sql` soubor (např. `[v119_multi_tenancy.sql](file:///...)`).
3. **Verzování**: Soubory musí striktně dodržovat verzování `v[číslo]_[název].sql`.
4. **Návod**: Vždy přidej stručný návod, kam přesně má uživatel kód vložit (např. Supabase SQL Editor).
